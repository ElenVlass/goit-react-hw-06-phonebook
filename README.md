Redux & Redux Toolkit

Приложение для хранения контактов телефонной книги.
Управление состоянием при помощи библиотеки Redux.

Приложение состоит из формы и списка контактов.

Хранение контактов телефонной книги в localStorage.
Используются методы жизненного цикла.

При добавлении и удалении контакта, контакты сохраняются в локальное хранилище.
При загрузке приложения, контакты, если таковые есть, считываются из локального хранилища и записываются в состояние.
